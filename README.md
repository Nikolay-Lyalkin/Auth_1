Запуск осуществляется посредством docker-compose.

При первой инициализации проекта создаётся superuser и роль superuser:
password=superuser
email=superuser@mail.ru
login=superuser
first_name=superuser
last_name=superuser

Пользователь с ролью superuser имеет доступ ко всем ручкам, не нарушающим жизнь иных пользователей.

Внимательно изучите .env.samle. Необходимо добавить две переменные необходимые для авторизации через OAUTH 2.0.
Будьте осторожны, при выходе, токен отправляется в кэш в редис до конца срока действия, повторно с этим же токеном обратиться к ручкам не получится.
